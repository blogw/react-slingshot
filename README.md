# React-Demo

## Machine Setup

1. **Install [Node](https://nodejs.org)**

2. **Install [Git](https://git-scm.com/downloads)**

3. **Install [VS Code](https://code.visualstudio.com/)**

4. **Install VS Code Plugin**
    - ESLint
    - Prettier-Code formatter
    - EditorConfig for VS Code

## Start

1. npm install
2. npm run start
